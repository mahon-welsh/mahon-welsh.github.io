When I was Stuck on Something Simple

I’d say the simplest problem I was stuck on this week was during the javascript-kata suite 2 challenge. I was having a rough time with the syntax of the part where I had to add the children to the children object, and assign it to Terah. The way I cracked it definitely relied on the rubber duck method discussed in the unit. I realised that I was basically just creating a new object with a value ben, instead of using curly brackets to add the actual ben object. When saying what I wanted the code to do aloud, I realised exactly what the issue was because that line didn’t make sense when turning the code into a sentence. I was really frustrated because I was certain everything looked right, and the console log looked super similar to what it was trying to get me to input, but once I figured everything out I was really stoked. I learnt the value of using console log to test stuff out, as well as a lot of helpful stuff about the correct brackets and such to use in situations thanks to all the stuff I googled trying to figure it out.
