Classes Vs. Ids, which should I use?

Pros of Classes

A "class" is a set of styling elements that can be applied to different html elements in order for said tag, as well as any elements inside the parent element, to adopt the styles and rules of the class. Classes are useful because they can be applied to any number of tags within an HTML document.

Pros of Ids

An "Id" has a similar function to a class in that it is a group of styling elements to be applied to an html element, however, an Id can only be applied to one element. For example, classes are best used when many elements need to function similarly or share a format. An Id should be used when an element needs to posess a unique function.
